서비스 계약 문서
===================

1. 프로젝트 관리 서비스 (Project Management Service)
--------------------------------------------------
**역할:**
    프로젝트의 생성, 수정, 삭제, 참여 관리

**기능:**
    - 프로젝트 생성 및 삭제
    - 프로젝트 정보 수정 (이름, 설명 등)
    - 프로젝트 멤버 추가 및 제거
    - 멤버 역할 변경 (Guest ↔ Member ↔ Manager)
    - 프로젝트 탈퇴 기능

2. 에픽 및 태스크 관리 서비스 (Epic & Task Management Service)
---------------------------------------------------------
**역할:**
    에픽과 태스크의 생성 및 관리, 마감일 변경에 따른 종속 관계 유지

**기능:**
    - 에픽 생성, 수정, 삭제
    - 태스크 생성, 수정, 삭제
    - 태스크 할당 및 담당자 변경
    - 에픽 마감일 변경 시 하위 태스크 조정

3. Urgent 및 우선순위 관리 서비스 (Urgency & Priority Service)
----------------------------------------------------------
**역할:**
    마감일을 기준으로 Urgent 값 자동 조정 및 Priority 변경 로직 관리

**기능:**
    - 마감일 기반 Urgent 값 자동 계산
    - Priority 변경 로직 (현재 미정)
    - UI 하이라이팅 및 알림 시스템과 연계

4. 알림 및 커뮤니케이션 서비스 (Notification & Communication Service)
---------------------------------------------------------------
**역할:**
    사용자 간의 소통을 돕고 중요한 변경 사항을 알림

**기능:**
    - 프로젝트 내 채팅 및 댓글 기능 제공
    - Urgent 값 변경 시 알림 발송
    - 마감일 임박 알림 (예: 24시간 전)
    - 역할 변경 및 태스크 할당 알림

5. 외부 연동 서비스 (External Integration Service)
----------------------------------------------------
**역할:**
    외부 시스템과의 연계를 담당

**기능:**
    - 캘린더 (Google, Outlook) 연동
    - 이메일 및 푸시 알림 서비스
    - 기타 API 연계 (향후 확장 가능)
